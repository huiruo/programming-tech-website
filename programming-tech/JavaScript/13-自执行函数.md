## 例子，下面形成了闭包
参考：[步骤2-作用域-块级作用域原理-作用域和闭包](../解析-解释-编译-事件循环-promise-安全/步骤2-作用域-块级作用域原理)
```js
function func() {
  var a = 1;
  (function () {
    console.log(a)
  }());
}
func(); // 1
```

## 两种写法
```js
// 写法1
(function func1(){
    console.log("func1");
})();

// 写法2
(function func2(){
    console.log("func2");
}()); 
```
