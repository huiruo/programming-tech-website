
#### 回文
```
左右读都一样

字串：原始字符串的一个连续子集
子序列：原始字符串的一个子集
```

#### 动态规划解题
```
性质：一个回文去掉两头，剩下的还是回文 (长度限制2)

状态： dp[i][j] 表示子串s[i..j]是否为回文
得到状态转移方程：
dp[i][j] = (s[i] == s[j]) and dp[i + 1] [j-1]
边际条件：
j - 1 - (i + 1) + 1 < 2 ,整理得j - i <3 <===> j - i + 1<4
s[i ..j]长度为2或则3时候，不用检查子串是否回文
```

每一步计算的结果都利用了之前计算的结果
快在利用状态转移方程快速得到一个子串是否回文串
